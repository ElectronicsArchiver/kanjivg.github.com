Related projects

This page gives a list of some other open-source projects which are similar to KanjiVG in scope.

  • The GlyphWiki uses a dedicated language and generation engine to describe and generate kanji.
  • Make Me a Hanzi contains free, open-source Chinese character data
  • AnimCJK - Draw animated Japanese characters (Kanji and Kana) and Chinese characters (Hanzi) in correct stroke order using svg, free open-source code.
