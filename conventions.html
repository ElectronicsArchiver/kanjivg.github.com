Conventions of KanjiVG

This page details some choices made by KanjiVG. If you wish to discuss changing the individual choices, please use the mailing list (see the link on the left) to bring up your proposal.

Radicals used

This section details the choices made for the values of kvg:element, the visual representation of a kanji radical.

Sanzui

The radical used for sanzui elements (the variant form of Kangxi radical 85, "water") is U+6C35 氵 rather than U+2EA1 ⺡, except for the special case of U+2EA1 itself.

Kemono-hen

The radical used for kemono-hen elements (the variant form of Kangxi radical 94, "dog") is U+2EA8 ⺨ rather than U+72AD 犭, except for the special case of U+72AD itself.

Shin-nyō

The radical used for shin-nyō elements (Kangxi radical 162, "walk") is either U+8FB6 辶 or U+2ECC ⻌ depending on whether the radical consists of four or three elements respectively. (Confusingly, the visual display of U+8FB6 in a font may appear to contain only three strokes.)

The visually highly similar U+2ECD, from CJK Radicals Supplement, and U+FA66, from CJK Compatibility Ideographs, are not used by KanjiVG.

Shoku-hen

飠 U+98E0 from CJK Unified Ideographs is used for the eight-stroke variant of shoku-hen, but ⻞ U+2EDE from CJK Radicals Supplement is used for the nine-stroke variant, since there does not seem to be a visually-close equivalent in CJK Unified Ideographs.

Other radicals

For all of the non-variant radical forms, there are two equivalent Unicode code points with the same visual form. One is in the Kangxi Radicals block of Unicode, from U+2F00 to U+2FD5, and the other is in the CJK Unified Ideograph block of Unicode. The equivalence between these is given in the Unicode Character Database (UCD) in the file CJKRadicals.txt, which maps between the pairs.

KanjiVG uses only the CJK Unified Ideograph versions of these equivalent entries. KanjiVG does not use the Kangxi Radicals block at all.

However, KanjiVG does use some parts of the CJK Radicals Supplement block from U+2E80 to U+2EF3, for example U+2E97 ⺗ is used for the shitagokoro variant of Kangxi radical 61, rather than the CJK Unified Ideographs equivalent form U+38FA 㣺.
