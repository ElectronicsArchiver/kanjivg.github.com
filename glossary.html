Glossary

This glossary consistently uses the word kanji to indicate sinographs, Chinese characters, hanzi, and so forth.

CJK Unified Ideographs

A block of Unicode containing kanji. The basic block goes from U+4E00 to U+9FFF, but it was extended with CJK Unified Ideographs Extension A from U+3400 to U+4DBF.

CJK Strokes

A block of Unicode containing kanji stroke shapes.

CJK Radicals

A block of Unicode containing kanji radicals.

Han Unification

Part of the original effort of Unicode to fit as many characters as possible into sixteen bits by using the same code point for characters considered to be identical. These characters are often graphically different, thus the Japanese-style KanjiVG vector graphics for various Unicode code points are not appropriate for Chinese usage.

JIS Level One and Two

第1・第2水準漢字だいいち・だいにすいじゅんかんじ Two sets of kanji which were used as the basis for the coverage of KanjiVG.

These sets were the standard sets of kanji available on Japanese-language computing platforms before the advent of Unicode.

Kangxi radicals

The historical radical system of an old Chinese dictionary. These radicals are labelled general or tradit in KanjiVG, in contrast with the Nelson radicals.

Kangxi Radicals Block

A block of Unicode containing the canonical form of the 214 Kangxi radicals.

Nelson radicals

The altered radicals used in the Nelson kanji dictionary. Nelson used a more systematic approach to recategorise kanji into radicals, resulting in different values than the Kangxi radical system.
