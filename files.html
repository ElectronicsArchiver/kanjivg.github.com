KanjiVG files

The KanjiVG file format is an extension of SVG. All KanjiVG files are 100% SVG-compliant, and can be opened by any SVG viewer or editor as-is. The shape of each kanji stroke (an individual line) is described as an SVG path on an area of size 109x109 pixels. Since these are vector graphics, they can be rescaled at will to any convenient size.

The SVG file is organized to reflect both the structure and stroke order of the kanji. Furthermore, several additional attributes provide extra information about the kanji's structure. See SVG Format for details.

Character models

The main graphical data is is based on schoolbook fonts (教科書体きょうかしょたい), which is a variation of the calligraphic kaisho style (楷書かいしょ). The exact fonts used as models for the KanjiVG graphical data are listed on the References page.

There is data for variations of characters and stroke order. When characters in schoolbook style deviate from traditional kaisho, there might be an extra version for kaisho, labelled with the suffix -Kaisho. There are also different versions for different stroke orders. See Variants for more information on the file naming and what variants exist.
